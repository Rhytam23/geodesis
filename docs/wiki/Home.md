# 🌾 Geodesis Wiki

Welcome to the **Geodesis** GitHub Wiki!

Geodesis is an open-source **Geospatial Digital Twin Platform** designed for climate-resilient agriculture and environmental adaptation planning in the Vietnamese Mekong Delta.
