# Installation Guide

## Prerequisites
- **Node.js**: `>= 20.0.0`
- **npm**: `>= 10.0.0`
- **Git**: `>= 2.30.0`

## Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-org/geodesis.git
   cd geodesis
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure environment variables (optional)**:
   ```bash
   cp .env.example .env.local
   ```

4. **Start the local development server**:
   ```bash
   npm run dev
   ```

5. **Open in browser**:
   Navigate to [http://localhost:3000/workspace](http://localhost:3000/workspace).
